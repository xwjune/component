> 倒计时

## API

| Name | Description | Type | Default |
| :--- | :--- | :--- | :--- | :--- | :--- |
| src | 图片资源 | string | - |
| blanksrc | 空白页图片资源 | string | - |
| onError | 图片加载失败回调 | function | - |
| ... | 其他图片属性 | any |

```jsx
<ImgCatch
  width="100"
  height="100"
  src="pic.png"
  blanksrc="blanksrc.png"
/>
```
